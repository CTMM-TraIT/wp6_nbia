### README.txt ###


This directory contains the configuration files for nbiaws.esciencetest.nl

The configuration files below contain the settings for the Apache webservice, the reverse proxy setup and the Shibboleth
configuration files needed for the SURFconext setup. 

Note that these configuration files are an example only and that settings may vary between operation systems and versions. 

Locations of configuration files
================================ 

Apache: 

- /etc/httpd/conf.d/nbiaws-ssl.conf

Shibboleth:  

- /etc/shibboleth/attribute-map.xml

- /etc/shibboleth/shibboleth2.xml
